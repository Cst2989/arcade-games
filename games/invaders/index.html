Open Source Invaders

Open Source Invaders is a free, open-source browser arcade game that turns any public GitHub repository into a retro Space Invaders experience. Enter a repo like facebook/react or vitejs/vite and the top 5 contributors become the five levels, with the most active committer as the final boss.

How to play

  • Move with arrow keys or A / D
  • Fire with Space
  • Drop a bomb with X
  • Pause with Escape

Share a level

Share any battle by appending ?repo=owner/name to the URL — for example, ?repo=tanstack/query. Friends will land straight into the briefing screen for that repo.
